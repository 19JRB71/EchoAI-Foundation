Here's Prompt 47: Advanced ROI Dashboard for EchoAI:
EchoAI is an AI-powered SaaS marketing platform. You've built the complete system including AI Email Marketing Campaigns. Now you need to add an Advanced ROI Dashboard that shows business owners exactly how much revenue EchoAI is generating for their business with clear dollar attribution across every channel — Facebook ads, SMS campaigns, email campaigns, phone calls, appointments, and lead conversions.
In the prompts folder create an ROI analysis prompt file that defines the AI ROI Analyst Agent. This agent accepts a brand profile and a complete performance dataset spanning all channels for a given time period. It analyzes the data, calculates true return on investment per channel, identifies the highest performing and lowest performing channels, spots trends and patterns in what is driving revenue, and generates a plain language executive summary that a non-technical business owner can understand and act on. The summary includes three specific recommendations for where to invest more and where to cut back based on the actual data.
In the controllers folder create an ROI dashboard controller with the following functions. First a get ROI summary function that accepts a brand ID and date range, pulls performance data from every channel in the platform including Facebook campaigns, SMS campaigns, email campaigns, phone calls, appointments booked, leads converted, and weekly analytics, calculates total spend across all paid channels, total revenue attributed to converted leads, cost per lead by channel, cost per appointment by channel, cost per conversion by channel, and overall ROI as a percentage, and returns a unified summary. Second a get channel breakdown function that returns detailed performance metrics for each individual channel side by side so the business owner can compare them directly. Third a get revenue attribution function that traces each converted lead back to the original channel that first touched them, showing which channel deserves credit for the conversion. Fourth an AI analysis function that takes the complete ROI dataset and calls the Anthropic API with the ROI analyst agent to generate the plain language executive summary and recommendations. Fifth a get historical ROI function that returns ROI data week by week for the last twelve weeks so the business owner can see their trajectory over time.
In the models folder create a migration file that adds an ROI snapshots table with columns for snapshot ID, brand ID, period start, period end, total spend, total revenue, total leads, total conversions, total appointments, ROI percentage, channel breakdown as JSON, AI analysis text, and timestamps.
Wire the ROI snapshot generation into the existing Monday morning scheduler so a new ROI snapshot is automatically calculated and saved every week for every active brand. This gives business owners a running history of their performance without having to manually request it.
In the client source folder update the existing ROI Dashboard section that is already in the sidebar. Replace the placeholder content with the full advanced dashboard. The dashboard has four tabs.
Tab one is the Overview showing the most important numbers at a glance. Show four large metric cards for total spend this period, total revenue attributed, overall ROI percentage displayed in large bold text colored green if positive and red if negative, and total conversions. Below the metric cards show the AI executive summary in a clean card with a refresh button to regenerate the analysis. Below that show a twelve week ROI trend line chart.
Tab two is Channel Breakdown showing a side by side comparison table of every channel with their spend, leads generated, appointments booked, conversions, cost per lead, cost per conversion, and ROI. Color code each channel row with green for positive ROI and red for negative ROI. Include a simple bar chart showing leads generated per channel.
Tab three is Revenue Attribution showing a visual funnel from leads to appointments to conversions with drop-off rates at each stage per channel. Show which channel has the best lead to conversion rate and which has the worst.
Tab four is History showing a table of weekly ROI snapshots for the last twelve weeks with their key metrics. Allow clicking a week to see the full breakdown for that period.
Show a date range selector at the top of the dashboard that lets the business owner filter all data to last seven days, last thirty days, last ninety days, or a custom range.
Apply enterprise tier gate to the advanced ROI dashboard routes since this is an Enterprise feature. The existing basic ROI dashboard placeholder visible to lower tiers should show an upgrade prompt explaining what they are missing.
Rebuild the React client after making changes. Output all files created and updated.
