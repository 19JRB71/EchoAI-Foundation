Alright, here's Prompt 5: Facebook Ad Campaign Agent for EchoAI:
EchoAI is an AI-powered SaaS marketing platform. You've already built the project structure, database schema, authentication, and Stripe billing. Now you need to wire up the Facebook Ad Campaign Agent that automates ad creation, management, and optimization.
In the config folder, create a Facebook configuration file that initializes the Facebook Business SDK using the Facebook App ID, App Secret, and Access Token from environment variables.
In the controllers folder, create a campaign controller with the following functions. First, a connect Facebook account function that accepts a Facebook ad account ID from the user, verifies the connection using the Facebook API, and stores the credentials securely in the api integrations table. Second, a create campaign function that accepts campaign goals, budget, and target audience details, uses the Facebook API to create a new ad campaign, ad set, and initial ad creative, stores the campaign record in the campaigns table, and returns the campaign ID. Third, an optimize campaign function that pulls performance data from Facebook for all active campaigns, analyzes cost per lead and conversion rate, adjusts bids and audiences based on what's working, and updates the campaigns table with the latest metrics. Fourth, a get campaign performance function that returns all active campaigns with their current performance metrics. Fifth, a generate ad creative function that accepts the brand profile from the brands table and uses it to generate ad copy and image prompts tailored to that brand's voice and personality.
In the utils folder, create a Facebook API helper file with reusable functions for making authenticated calls to the Facebook Graph API, handling rate limits, and managing API errors gracefully.
In the routes folder, create a campaign routes file with POST routes for connecting a Facebook account and creating campaigns, GET route for campaign performance, POST routes for optimizing campaigns and generating ad creative. All routes should be protected with auth middleware and lockout middleware.
Update server dot js to include the campaign routes at slash api slash campaigns. Output all files created.
