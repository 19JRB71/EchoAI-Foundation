Here's Prompt 31: Sales Script Generator for EchoAI:
EchoAI is an AI-powered SaaS marketing platform. You've built the complete system including the AI Website Chatbot. Now you need to add a Sales Script Generator that creates customized sales scripts for business owners and their teams to use on phone calls, in person meetings, and follow up conversations.
In the prompts folder create a sales script prompt file that defines the AI Sales Script Agent. This agent accepts a brand profile, the type of sale, the target customer persona, common objections the business faces, and the desired outcome of the conversation. It generates a complete sales script including a strong opening that builds rapport immediately, discovery questions that uncover the prospect's pain points and goals, a pitch section that connects the product or service directly to what the prospect just shared, objection handling responses for the five most common objections the business faces, a closing section with three different closing techniques from soft to direct, and a follow up sequence with suggested messages for day one, day three, and day seven after the call. The script should sound completely natural and human, never robotic or salesy. It should match the brand's voice and personality exactly.
In the controllers folder create a sales script controller with the following functions. First a generate script function that accepts brand ID, sale type, target persona, common objections, and desired outcome, calls the Anthropic API with the sales script agent prompt, and returns the complete sales script. Second a save script function that saves a generated script to a new sales scripts table. Third a get scripts function that returns all saved scripts for a brand. Fourth an update script function that allows editing a saved script. Fifth a delete script function that removes a saved script.
In the models folder create a migration file that adds a sales scripts table with columns for script ID, brand ID, sale type, target persona, full script content as JSON, status as draft or active, and timestamps.
In the client source folder add a Sales Scripts section to the existing dashboard sidebar. This section has two tabs. Tab one is the Script Generator. Show a form where users select their brand, enter the type of sale they are making such as cold call, warm follow up, or in person meeting, describe their ideal target customer, list their three to five most common objections, and describe their desired outcome. When they click Generate Script the AI creates the full script and displays it in a clean readable format with clearly labeled sections for opening, discovery, pitch, objection handling, closing techniques, and follow up sequence. Show a Save Script button and a Copy to Clipboard button. Tab two is Saved Scripts showing all saved scripts with their sale type, target persona, and date created. Clicking a script opens it in full view with an edit option.
Wire all endpoints to the backend. Rebuild the React client after making changes. Output all files created.
