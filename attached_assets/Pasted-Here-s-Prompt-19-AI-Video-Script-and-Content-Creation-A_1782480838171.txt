Here's Prompt 19: AI Video Script and Content Creation Agent for EchoAI:
EchoAI is an AI-powered SaaS marketing platform. You've built the complete system including social media management and posting. Now you need to add an AI Video Script and Content Creation Agent that helps business owners create compelling video content for their social media platforms, Facebook ads, and YouTube channel.
In the prompts folder create a video content prompt file that defines the AI Video Content Agent. This agent accepts a brand profile, a video topic or goal, the target platform, and the desired video length. It generates a complete video package including a hook that grabs attention in the first three seconds, a full script broken into scenes with suggested visuals for each scene, on-screen text overlays and captions, a strong call to action at the end, suggested background music style, and a thumbnail concept description. The script should match the brand's voice and personality exactly. For TikTok and Instagram Reels it generates short punchy scripts under sixty seconds. For YouTube it generates longer educational or entertaining scripts up to ten minutes. For Facebook it generates ad-style scripts optimized for conversions.
In the controllers folder create a video content controller with the following functions. First a generate video script function that accepts brand ID, video topic, target platform, and desired length, calls the Anthropic API with the video content agent prompt, and returns the complete video package. Second a save script function that saves a generated script to a new video scripts table so users can come back to it later. Third a get scripts function that returns all saved scripts for a brand. Fourth a delete script function that removes a saved script.
In the models folder create a migration file that adds a video scripts table with columns for script ID, brand ID, platform, topic, video length, full script content as JSON, status as draft or published, and timestamps.
In the client source folder add a new Video Content section to the existing dashboard sidebar. This section has two tabs. Tab one is the Script Generator. Show a form where users select their brand, enter a video topic, select the target platform, and choose a video length from short under sixty seconds, medium one to three minutes, or long five to ten minutes. When they click Generate Script the AI creates the full video package and displays it in a clean readable format showing the hook, each scene with its script and visual description, on screen text suggestions, call to action, music suggestion, and thumbnail concept. Show a Save Script button and a Copy to Clipboard button. Tab two is Saved Scripts. Show all saved scripts in a list with their topic, platform, length, and date created. Clicking a script opens it in full view.
Wire all endpoints to the backend. Rebuild the React client after making changes. Output all files created.
