Here's Prompt 43: Sidebar Redesign and Visual Overhaul for EchoAI:
EchoAI is an AI-powered SaaS marketing platform. You've built a complete system with 42 prompts worth of features. The sidebar has grown long and overwhelming. Now you need to redesign the sidebar navigation into a clean organized grouped structure with tier-based color coding so the dashboard feels premium, intuitive, and easy to navigate.
The three tier accent colors are as follows. Starter tier features use blue as the accent color with hex value 3B82F6. Professional tier features use purple as the accent color with hex value 8B5CF6. Enterprise tier features use gold as the accent color with hex value F59E0B. These colors are used for active nav item highlights, lock badge backgrounds, section group headers, and upgrade prompt cards throughout the dashboard.
Redesign the sidebar navigation into five clearly labeled collapsible groups. Each group has a small group header label and contains related nav items. The groups are as follows.
Group one is called Overview and contains Dashboard and Leads. This group is always expanded by default and has no tier color since these are core features.
Group two is called Marketing and contains Campaigns, Ad Studio, Social Media, Content Calendar, Email Marketing, Video Content, and Follow-Up Sequences. This group uses a mix of Starter blue for Campaigns and Professional purple for the rest.
Group three is called Customer Management and contains Phone Agent, Website Chatbot, Appointments, Reputation, Feedback, and Zapier. Phone Agent, Website Chatbot, and Appointments use Professional purple. Feedback uses Enterprise gold. Zapier uses Professional purple.
Group four is called Content and Tools and contains Sales Scripts, Image Studio, Google and SEO, and Video Content. Sales Scripts uses Professional purple. Image Studio and Google and SEO use Professional purple.
Group five is called Business and contains ROI Dashboard, Affiliate Program, White Label, and Settings. ROI Dashboard uses Professional purple. Affiliate Program and White Label use Enterprise gold. Settings has no tier color.
Admin nav item stays at the very bottom outside all groups and is only visible to admin users.
Each nav item that is locked for the current user's tier shows a small colored badge with the tier name using the tier's accent color. The badge says PRO in purple or ENT in gold. Clicking a locked item shows the upgrade prompt instead of the feature.
Active nav items highlight with a left border and background tint using the item's tier color. Starter items highlight in blue. Professional items highlight in purple. Enterprise items highlight in gold. Items with no tier color highlight in the existing teal accent color.
The groups are collapsible. Clicking a group header toggles it open or closed. The group that contains the currently active section is always expanded. All other groups start collapsed on first load to keep the sidebar compact. Store the expanded state in localStorage so it persists between sessions.
Each group header shows a subtle indicator of how many features in that group are locked for the current user's tier so they can see at a glance what they are missing.
Update the overall dashboard visual design at the same time. Make the main content area cards and metric boxes use subtle left border accents in the tier color of the feature being viewed. For example viewing the Reputation section which is a Professional feature shows subtle purple accents on the cards. Viewing an Enterprise feature shows gold accents. Viewing a Starter feature shows blue accents. This reinforces the tier association throughout the experience.
Update the FeatureGate upgrade prompt cards to use the tier color of the required plan. Professional upgrade prompts use purple gradient backgrounds. Enterprise upgrade prompts use gold gradient backgrounds. Make these cards visually compelling and aspirational.
Update the top navigation bar to show the current user's tier as a colored badge next to their name. Starter shows a blue badge. Professional shows a purple badge. Enterprise shows a gold badge. Admin shows a special badge in the existing teal color.
Make the entire sidebar mobile responsive. On mobile screens the sidebar collapses to a bottom navigation bar showing only icons for each group. Tapping a group icon expands that group's items in a slide-up panel.
Rebuild the React client after all changes. Output all files created and updated.
