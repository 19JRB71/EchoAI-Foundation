Here's Prompt 37: AI Ad Creative Studio for EchoAI:
EchoAI is an AI-powered SaaS marketing platform. You've built the complete system including the AI Content Calendar. Now you need to add an AI Ad Creative Studio that generates complete ad creatives — images, video scripts, ad copy, and full campaign packages — tailored specifically to each business using the deep brand understanding built during the brand discovery process.
In the prompts folder create an ad creative studio prompt file that defines the AI Ad Creative Director Agent. This agent has full access to the brand profile built during the three part brand discovery process including the business owner's personality, voice, visual preferences, target audience, unique selling proposition, and competitive positioning. It uses this deep brand understanding to generate ad creatives that feel authentically produced by that specific business, not generic templates. The agent generates five distinct creative concepts per request, each with a different angle — emotional, logical, social proof, urgency, and curiosity. For each concept it produces a complete ad package including a primary headline under ten words that stops the scroll, three body copy variations from short punchy to detailed, a visual description so detailed that a designer or AI image generator could recreate it exactly, a video script broken into hook, story, and call to action, suggested audience targeting based on the brand's ideal customer profile, recommended ad placement such as Facebook feed, Instagram stories, or TikTok, estimated performance tier based on how well the creative matches proven ad frameworks, and a split test recommendation pairing two concepts that would produce the most useful performance data.
In the controllers folder create an ad creative studio controller with the following functions. First a generate creatives function that accepts brand ID, campaign goal such as lead generation, sales, or brand awareness, budget range, and an optional specific product or service to promote, pulls the complete brand profile and discovery session data from the database, calls the Anthropic API with the full brand context and the ad creative director prompt, and returns five complete creative packages. Second a save creative function that saves a generated creative package to a new ad creatives table. Third a get creatives function that returns all saved creatives for a brand organized by campaign goal. Fourth a launch creative function that takes a saved creative and creates a real Facebook ad campaign using the existing Facebook API integration, using the creative's headline, body copy, audience targeting, and placement recommendations. Fifth a get creative performance function that pulls real performance data from Facebook for launched creatives and updates the creative record with actual results.
In the models folder create a migration file that adds an ad creatives table with columns for creative ID, brand ID, campaign goal, creative concept as JSON containing all five packages, status as draft, launched, or archived, facebook campaign ID if launched, performance data as JSON, and timestamps.
In the client source folder add a new AI Ad Studio section to the existing dashboard sidebar. This section has three tabs.
Tab one is the Creative Generator. Show a form where users select their brand, choose their campaign goal from a dropdown, enter their budget range, and optionally describe a specific product or service they want to promote. When they click Generate Creatives the AI produces five complete creative packages. Display each package as an expandable card showing the concept angle, headline, body copy variations, visual description, video script, audience targeting, recommended placement, performance tier prediction, and split test recommendation. Each card has a Save Creative button and a Launch to Facebook button.
Tab two is the Creative Library showing all saved creatives organized by campaign goal with their status, launch date if applicable, and performance metrics if the creative has been launched. Clicking a creative shows its full package.
Tab three is Performance showing all launched creatives with their real Facebook performance data including impressions, clicks, cost per click, leads generated, and cost per lead. Sort by best performing first. Include a comparison view showing which creative angles perform best for that specific brand so the AI can learn and improve future recommendations.
Wire the launch creative function into the existing Facebook campaign infrastructure so creatives can go live with one click. Wire the get creative performance function into the existing Monday morning scheduler so performance data updates automatically every week alongside campaign optimization.
Rebuild the React client after making changes. Output all files created.
