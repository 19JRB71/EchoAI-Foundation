Here's Prompt 21: AI Image Generation for Ads and Social Posts for EchoAI:
EchoAI is an AI-powered SaaS marketing platform. You've built the complete system including social media management, video scripts, and email campaigns. Now you need to add an AI Image Generation system that creates professional marketing images for Facebook ads, social media posts, and email campaigns directly inside EchoAI.
In the controllers folder create an image generation controller with the following functions. First a generate image function that accepts a brand ID, image purpose such as Facebook ad, Instagram post, or email header, a description of what the image should show, and the brand's visual style preferences pulled from the brand profile, then calls the OpenAI DALL-E API to generate a professional marketing image and returns the image URL. Second a generate ad creative set function that accepts a brand ID and campaign goal and automatically generates a set of three image variations for A/B testing — each with slightly different visual approaches but consistent branding. Third a save image function that saves a generated image URL and its metadata to a new images table. Fourth a get images function that returns all saved images for a brand organized by purpose. Fifth a delete image function that removes a saved image.
In the prompts folder create an image prompt builder file that constructs optimized DALL-E prompts from the brand profile and image requirements. It should factor in the brand's visual style preferences, color preferences, target audience, and the specific platform requirements such as square for Instagram, landscape for Facebook ads, and portrait for Stories. The prompts should produce clean professional marketing images not amateur looking graphics.
In the models folder create a migration file that adds an images table with columns for image ID, brand ID, purpose, prompt used, image URL, platform, status as saved or used, and timestamps.
In the client source folder add a new Image Studio section to the existing dashboard sidebar. This section has two tabs. Tab one is the Image Generator. Show a form where users select their brand, choose the image purpose from Facebook Ad, Instagram Post, Twitter Post, LinkedIn Post, Email Header, or YouTube Thumbnail, enter a description of what they want, and click Generate. Show three generated image variations side by side. Each image has a Save Image button, a Use in Social Post button that pre-fills the social media scheduler with that image, and a Download button. Tab two is the Image Library showing all saved images organized by purpose with filtering. Each image shows a thumbnail, its purpose, date created, and options to download or delete.
Wire all endpoints to the backend using the existing OpenAI API key already configured. Rebuild the React client after making changes. Output all files created.
