Here's Prompt 24: Facebook OAuth Connection Flow for EchoAI:
EchoAI is an AI-powered SaaS marketing platform. You've built the complete system. Now you need to build the proper Facebook OAuth connection flow so customers can securely connect their Facebook ad account with just a few clicks instead of manually entering API tokens.
In the backend create a Facebook OAuth controller with the following functions. First an initiate OAuth function that generates a Facebook OAuth URL with the correct permissions including ads management, ads read, business management, and pages read engagement, stores a state parameter in the session to prevent CSRF attacks, and redirects the user to Facebook to authorize. Second an OAuth callback function that accepts the authorization code from Facebook, exchanges it for a long lived access token, fetches the user's Facebook ad accounts, saves the access token and ad accounts to the api integrations table encrypted, and redirects the user back to their EchoAI dashboard with a success message. Third a get connected accounts function that returns all connected Facebook ad accounts for the user without exposing the actual tokens. Fourth a disconnect account function that removes a Facebook integration from the database.
Add express session middleware to handle the OAuth state parameter securely. Add the connect-pg-simple package to store sessions in PostgreSQL so they persist across server restarts.
In the models folder create a migration file that adds a sessions table for PostgreSQL session storage and adds a facebook ad accounts column to the api integrations table to store the list of ad accounts as JSON.
In the client source folder update the Settings section Facebook connection area to use the new OAuth flow. Instead of asking users to manually paste a token show a Connect with Facebook button that uses the official Facebook blue color and logo. When clicked it redirects them to Facebook to authorize. When they return show their connected ad accounts in a dropdown so they can select which account EchoAI should manage. Show a green connected badge next to each connected account and a disconnect button.
Also update the onboarding wizard Step two Facebook connection to use this same OAuth button flow instead of any manual token entry.
Wire all OAuth endpoints. Rebuild the React client. Output all files created.
