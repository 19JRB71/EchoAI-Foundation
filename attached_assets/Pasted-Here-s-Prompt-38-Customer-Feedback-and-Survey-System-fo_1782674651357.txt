Here's Prompt 38: Customer Feedback and Survey System for EchoAI:
EchoAI is an AI-powered SaaS marketing platform. You've built the complete system including the AI Ad Creative Studio. Now you need to add a Customer Feedback and Survey System that automatically collects reviews and feedback from customers after every interaction, analyzes sentiment, identifies patterns, and uses the insights to continuously improve the business owner's marketing and customer experience.
In the prompts folder create a feedback analysis prompt file that defines the AI Feedback Analyst Agent. This agent accepts a collection of customer feedback responses, analyzes sentiment as positive, neutral, or negative, identifies the top three recurring themes or patterns across all responses, highlights the most urgent issues needing attention, identifies what customers love most, and generates three specific actionable recommendations the business owner can implement immediately to improve their customer experience and marketing. The analysis should be written in plain language the business owner can understand and act on, not marketing jargon.
In the controllers folder create a feedback controller with the following functions. First a create survey function that accepts a brand ID and survey type such as post purchase, post call, post chatbot, or general, and generates a five question survey using the brand's voice and the specific interaction context. Second a send survey function that sends the survey to a customer via email or SMS after an interaction is completed. Third a record response function that saves a customer's survey response. Fourth an analyze feedback function that pulls all responses for a brand from the last thirty days, calls the Anthropic API with the feedback analyst agent, and returns a complete sentiment analysis and recommendations report. Fifth a get feedback dashboard function that returns aggregate stats including average satisfaction score, total responses, response rate, sentiment breakdown, and the latest AI analysis report. Sixth an auto send survey function that is called automatically after key interactions including after a lead chatbot conversation ends, after a phone call completes, and after a weekly report is sent, sending a short two question satisfaction survey to capture immediate feedback.
In the models folder create a migration file that adds a surveys table with columns for survey ID, brand ID, survey type, questions as JSON, and timestamps. Also add a survey responses table with columns for response ID, survey ID, brand ID, lead ID if applicable, respondent email, respondent phone, answers as JSON, sentiment score from one to ten, and timestamps. Also add a feedback reports table with columns for report ID, brand ID, analysis period start, analysis period end, total responses, average sentiment, themes as JSON, recommendations as JSON, full report text, and timestamps.
Wire the auto send survey function into the chatbot controller so a survey fires after every completed lead conversation. Wire it into the phone controller so a survey fires after every completed call. Wire it into the scheduler so a feedback analysis report is generated automatically every Monday alongside the weekly analytics report.
In the client source folder add a new Feedback and Reviews section to the existing dashboard sidebar. This section has three tabs. Tab one is the Feedback Dashboard showing average satisfaction score displayed as a large number with a color indicator green for eight or above, yellow for six to seven, red for below six, total responses this month, sentiment breakdown as a simple bar showing percentage positive neutral and negative, and the latest AI analysis report with themes and recommendations displayed as clean cards. Tab two is Responses showing all individual survey responses in a table with respondent info, their answers, sentiment score, interaction type, and date. Clicking a response shows the full answer. Tab three is Surveys showing all active surveys with their type, question count, response rate, and an option to customize the questions or create a new survey.
Rebuild the React client after making changes. Output all files created.
