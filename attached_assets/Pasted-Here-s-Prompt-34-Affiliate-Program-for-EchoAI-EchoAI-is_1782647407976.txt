Here's Prompt 34: Affiliate Program for EchoAI:
EchoAI is an AI-powered SaaS marketing platform. You've built the complete system including the White Label Agency System. Now you need to add an Affiliate Program that allows anyone to earn commissions by referring new customers to EchoAI. This creates a word of mouth sales army that grows the platform automatically.
In the controllers folder create an affiliate controller with the following functions. First a register affiliate function that accepts the authenticated user's application to become an affiliate, creates an affiliate record with a unique referral code, and returns their affiliate dashboard link. Second a get affiliate profile function that returns the affiliate's referral code, total referrals, total commissions earned, pending commissions, and paid commissions. Third a track referral function that is called when someone visits the EchoAI landing page with an affiliate referral code in the URL, stores the referral cookie, and attributes any subsequent signup to that affiliate. Fourth a convert referral function that is called automatically when a referred user completes their first payment, calculates the commission at twenty percent of the first month's subscription, and creates a commission record. Fifth a get commissions function that returns all commission records for an affiliate with their status as pending, approved, or paid. Sixth a request payout function that allows an affiliate to request payment of their approved commissions via PayPal email address. Seventh an admin approve commissions function that allows you as admin to approve pending commissions and mark them as paid.
In the models folder create a migration file that adds an affiliates table with columns for affiliate ID, user ID, referral code as a unique string, PayPal email, total earned, total paid, status as active or suspended, and timestamps. Also add a referrals table with columns for referral ID, affiliate ID, referred user ID, referral code used, commission amount, status as pending, approved, or paid, and timestamps.
In the utils folder create a referral tracking utility that generates unique referral codes, sets and reads referral cookies, and attributes signups to the correct affiliate.
Wire the convert referral function into the Stripe webhook handler so commissions are created automatically when a referred user's first payment succeeds.
In the client source folder add an Affiliate Program section to the existing dashboard sidebar. This section has three tabs. Tab one is the Dashboard showing the affiliate's unique referral link they can share, total clicks placeholder, total signups, total commissions earned, pending commissions, and a promotional materials section with suggested social media posts and email templates they can copy and share. Tab two is Commissions showing a table of all commissions with referred customer email, subscription tier, commission amount, and status. Tab three is Payouts showing their PayPal email setting and a Request Payout button that is enabled when they have approved commissions waiting.
Add an Affiliates management section to the admin panel showing all affiliates, their referral counts, total commissions, and buttons to approve commissions and suspend affiliates.
In the routes folder create affiliate routes with POST route for registering as affiliate, GET route for affiliate profile, GET route for commissions, POST route for requesting payout, and admin routes for approving commissions and listing all affiliates. Protect all routes with auth and lockout middleware.
Update the landing page to read the referral code from the URL and store it so it persists through the signup flow.
Update server dot js to include affiliate routes at slash api slash affiliates. Output all files created.
