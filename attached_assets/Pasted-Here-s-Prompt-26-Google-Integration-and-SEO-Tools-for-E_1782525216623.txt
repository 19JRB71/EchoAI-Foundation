Here's Prompt 26: Google Integration and SEO Tools for EchoAI:
EchoAI is an AI-powered SaaS marketing platform. You've built the complete system including Facebook ads, social media, email marketing, video scripts, and image generation. Now you need to add Google integration and SEO tools so customers can manage their entire digital marketing presence from one platform.
In the backend create a Google integration controller with the following functions. First a Google OAuth connection function that generates a Google OAuth URL with permissions for Google Business Profile, Google Ads, Google Analytics, and Google Search Console, handles the OAuth callback, exchanges the code for tokens, and saves them encrypted to the api integrations table. Second a Google Business Profile function that fetches the customer's Google Business Profile information, allows posting updates to their profile, and shows their review summary. Third a Google Ads function that connects to their Google Ads account, fetches active campaign performance, and allows creating simple search campaigns. Fourth a Google Analytics function that fetches their website traffic data including sessions, pageviews, bounce rate, and top traffic sources for the last thirty days.
In the prompts folder create an SEO content prompt file that defines the AI SEO Agent. This agent accepts a brand profile, a target keyword or topic, and a content type such as blog post, landing page, or product description. It generates fully SEO optimized content including a keyword-rich title, meta description under 160 characters, header structure with H1 H2 and H3 tags, full body content naturally incorporating the target keyword and related terms, internal linking suggestions, and an SEO score explanation telling the user why this content will rank well. The content should sound natural and human not keyword stuffed.
In the controllers folder create an SEO controller with the following functions. First a generate SEO content function that accepts brand ID, target keyword, content type, and calls the Anthropic API with the SEO agent prompt. Second a get keyword suggestions function that accepts a topic and returns ten related keyword ideas with estimated search volume categories as high medium or low. Third a save SEO content function that saves generated content to a new SEO content table. Fourth a get SEO content function that returns all saved SEO content for a brand.
In the models folder create a migration file that adds a Google integrations table for storing Google OAuth tokens and an SEO content table with columns for content ID, brand ID, keyword, content type, generated content as JSON, SEO score, and timestamps.
In the client source folder add a new Google and SEO section to the existing dashboard sidebar. This section has four tabs. Tab one is Google Connect showing connect buttons for Google Business Profile, Google Ads, and Google Analytics with OAuth connection status for each. Tab two is SEO Content Generator where users select their brand, enter a target keyword, choose a content type from blog post, landing page, or product description, and click Generate. The AI produces the full SEO optimized content displayed with the title, meta description, and full body in a clean readable format with a Save Content button and a Copy to Clipboard button. Tab three is Keyword Research where users enter a topic and get back ten keyword suggestions with their search volume category displayed as colored badges. Tab four is Google Analytics showing a simple dashboard with sessions, pageviews, bounce rate, and top traffic sources pulled from their connected Google Analytics account.
Wire all endpoints to the backend. Rebuild the React client after making changes. Output all files created.
