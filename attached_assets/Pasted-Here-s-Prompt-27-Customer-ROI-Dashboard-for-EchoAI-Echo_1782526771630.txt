Here's Prompt 27: Customer ROI Dashboard for EchoAI:
EchoAI is an AI-powered SaaS marketing platform. You've built the complete system. Now you need to add a Customer ROI Dashboard that shows each customer exactly how much value EchoAI is delivering to their business so they never want to cancel their subscription.
In the controllers folder create an ROI controller with the following functions. First a calculate ROI function that accepts a brand ID and pulls data from across the entire platform — total leads generated, hot leads count, estimated lead value based on industry averages, total ad spend managed, cost per lead improvement over time, social posts published, emails sent, campaigns run, hours saved estimate based on tasks automated, and money saved estimate compared to hiring a marketing agency. It calculates a total estimated ROI figure and returns a complete breakdown. Second a get ROI history function that returns weekly ROI snapshots for the last twelve weeks so customers can see their progress over time. Third a generate ROI report function that uses the Anthropic API to write a personalized monthly ROI summary in plain language telling the customer exactly what EchoAI did for their business that month, what results it achieved, and what it's working on next month.
In the prompts folder create an ROI report prompt file that generates a friendly, specific, results-focused monthly summary. It should feel like a report from a trusted marketing partner not a software tool. It should reference specific numbers, celebrate wins, acknowledge areas still improving, and build confidence that EchoAI is worth every dollar.
In the models folder create a migration file that adds an ROI snapshots table with columns for snapshot ID, brand ID, week date, total leads, hot leads, estimated lead value, ad spend managed, cost per lead, hours saved, money saved, total ROI estimate, and timestamps.
In the client source folder add a new ROI Dashboard section to the existing dashboard sidebar. This section has three parts. First a headline ROI summary card at the top showing four big numbers — total estimated value generated, hours saved, money saved versus hiring an agency, and return on investment as a percentage. Make these numbers large bold and visually impactful. Second a twelve week ROI trend chart showing how their results have improved week over week since joining EchoAI. Third a detailed breakdown section showing exactly where the value comes from — leads generated with estimated value, campaigns run with performance, social posts published with reach estimates, emails sent with open rates, and time saved on each task. At the bottom show a Generate Monthly Report button that produces the AI written monthly summary and allows downloading it as a PDF.
Wire all endpoints to the backend. Rebuild the React client after making changes. Output all files created.
