Here's Prompt 20: AI Email Marketing Campaign Agent for EchoAI:
EchoAI is an AI-powered SaaS marketing platform. You've built the complete system including social media management, video script generation, and Facebook ad automation. Now you need to add an AI Email Marketing Campaign Agent that helps business owners build email lists, create AI-generated email campaigns, and send automated email sequences to their leads directly from EchoAI.
In the prompts folder create an email marketing prompt file that defines the AI Email Campaign Agent. This agent accepts a brand profile, a campaign goal such as nurture leads, promote a product, or re-engage cold leads, a target audience description, and the number of emails in the sequence. It generates a complete email sequence where each email has a compelling subject line, preview text, full email body written in the brand's voice, a clear call to action, and the recommended send timing such as day one, day three, day seven. The copy should feel personal and human, not corporate. It should follow proven email marketing principles like open loops, storytelling, and value before ask.
In the controllers folder create an email campaign controller with the following functions. First a generate email sequence function that accepts brand ID, campaign goal, target audience, and number of emails, calls the Anthropic API with the email campaign agent prompt, and returns the complete email sequence. Second a save campaign function that saves the generated sequence to a new email campaigns table. Third a send campaign function that sends the next scheduled email in a sequence to a list of lead email addresses from the CRM. Fourth a get campaigns function that returns all email campaigns for a brand. Fifth a get campaign performance function that returns open rates, click rates, and unsubscribe rates for each campaign.
In the models folder create a migration file that adds an email campaigns table with columns for campaign ID, brand ID, campaign name, goal, email sequence as JSON, status as draft, active, or completed, and timestamps. Also add an email sends table that tracks each individual email sent with columns for send ID, campaign ID, lead ID, email address, subject, sent at, opened at, clicked at, and unsubscribed.
In the client source folder add a new Email Marketing section to the existing dashboard sidebar. This section has three tabs. Tab one is the Campaign Generator where users select their brand, enter a campaign goal, describe their target audience, choose how many emails to generate between three and ten, and click Generate. The AI creates the full sequence and displays each email as an expandable card showing subject line, preview text, body, call to action, and send timing. Show Save Campaign and Send to CRM Leads buttons. Tab two is Active Campaigns showing all saved campaigns with their name, status, number of emails, and a progress indicator showing how many emails have been sent. Tab three is Performance showing open rates, click rates, and unsubscribe rates per campaign in a simple clean table.
Wire all endpoints to the backend. Rebuild the React client after making changes. Output all files created.
