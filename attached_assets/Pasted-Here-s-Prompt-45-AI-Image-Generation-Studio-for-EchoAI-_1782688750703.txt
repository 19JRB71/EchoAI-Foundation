Here's Prompt 45: AI Image Generation Studio for EchoAI:
EchoAI is an AI-powered SaaS marketing platform. You've built the complete system including Two-Way SMS Marketing. Now you need to add an AI Image Generation Studio that creates professional marketing images, ad creatives, social media visuals, and brand graphics directly inside EchoAI using the brand's visual profile and preferences established during brand discovery.
In the prompts folder create an image generation prompt file that defines the AI Image Prompt Engineer Agent. This agent accepts a brand profile including visual style preferences, brand personality, target audience, and color preferences established during brand discovery, plus a specific image purpose such as Facebook ad, Instagram post, blog header, or logo concept, and a content description. It generates five highly detailed image generation prompts optimized for AI image generation that would produce professional marketing quality visuals perfectly aligned with the brand's aesthetic. Each prompt specifies the visual style, color palette, composition, mood, lighting, and any text overlay suggestions. The prompts are written to produce consistent on-brand visuals every time.
In the controllers folder create an image studio controller with the following functions. First a generate image prompts function that accepts a brand ID, image purpose, and content description, calls the Anthropic API with the image prompt engineer agent, and returns five detailed image generation prompts. Second a generate image function that accepts one of the generated prompts and calls the OpenAI DALL-E 3 API to generate the actual image, returning the image URL. Third a save image function that saves a generated image record to the database including the prompt used, the image URL, the purpose, and brand ID. Fourth a get images function that returns all saved images for a brand organized by purpose. Fifth a delete image function that removes a saved image record. Sixth a generate variations function that accepts an existing image prompt and generates three variations with slight style differences so the business owner has options to choose from.
In the models folder create a migration file that adds an image generations table with columns for image ID, brand ID, purpose, content description, ai prompt used, image URL, style notes, status as generated or archived, and timestamps.
In the client source folder add a new Image Studio section to the existing dashboard sidebar under the Content and Tools group. This section has three tabs.
Tab one is the Studio. Show a form at the top where users select their brand, choose the image purpose from a dropdown including Facebook Ad, Instagram Post, Instagram Story, TikTok Thumbnail, LinkedIn Post, Twitter Post, Blog Header, Email Header, and Logo Concept, and enter a brief content description of what they want. Show a Generate button. When clicked the system first generates five detailed AI image prompts shown as expandable cards. Each prompt card shows the style description and a Generate Image button. When they click Generate Image on a specific prompt it calls DALL-E 3 and displays the generated image. Show a Save Image button and a Download button on each generated image. Show a Generate Variations button that creates three more versions of that same prompt.
Tab two is the Image Library showing all saved images in a grid layout organized by purpose category. Each image shows its purpose badge, content description, and generation date. Clicking an image opens it full size with options to download, delete, or use in a social media post by sending it directly to the Social Media section.
Tab three is Brand Style Guide showing a summary of the brand's visual preferences pulled from the brand discovery profile including preferred colors, visual style, and mood. Show a Regenerate Brand Prompts button that refreshes the base image generation prompts based on the current brand profile.
Wire the Image Studio into the existing Ad Creative Studio so when generating ad creatives users can click Generate Image to create a matching visual for each creative concept directly from the Ad Studio section.
Wire the Image Studio into the Social Media section so when scheduling a post users can click Generate Image to create a matching visual for that post.
Apply professional tier gate to image studio routes since this is a Professional and above feature.
Rebuild the React client after making changes. Output all files created and updated.
