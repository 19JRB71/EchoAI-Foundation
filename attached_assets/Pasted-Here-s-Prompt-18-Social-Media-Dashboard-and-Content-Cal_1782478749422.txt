Here's Prompt 18: Social Media Dashboard and Content Calendar for EchoAI:
EchoAI is an AI-powered SaaS marketing platform. You've built the complete system including the multi-platform social media posting backend. Now you need to add the social media management section to the customer dashboard so users can see their content calendar, generate AI posts, schedule content, and track performance across all their connected social platforms.
In the client source folder add a new Social Media section to the existing dashboard. This section has four tabs.
Tab one is the Content Calendar. Show a monthly calendar view with all scheduled and published posts. Each day that has a post shows a colored dot indicating which platform it was posted to — blue for Facebook, pink for Instagram, black for TikTok, blue dark for LinkedIn, sky blue for Twitter, and red for YouTube. Clicking a day shows all posts scheduled for that day. Clicking a post shows its full content and status.
Tab two is the AI Content Generator. Show a simple form where the user selects their brand, enters a content topic or theme, selects which platforms they want to generate for, and clicks Generate. The AI generates five post variations for each selected platform. Display the results in a clean card layout with one card per variation. Each card shows the platform icon, the post content, suggested hashtags, and two buttons — Schedule This Post and Regenerate. When they click Schedule This Post show a date and time picker and save it to the queue.
Tab three is the Connected Accounts. Show which social platforms are connected and which are not. For each platform show a Connect button if not connected and a Disconnect button if connected along with the connected username. Keep this simple and clear.
Tab four is Performance. Show a simple table of all published posts with their platform, post date, and three engagement metrics — likes, shares, and reach. Sort by most recent first.
Add the Social Media section to the existing dark sidebar navigation with an appropriate icon. Make sure it fits seamlessly with the existing dashboard design — same dark sidebar, same light content area, same card and table styles already used throughout.
Wire all four tabs to the existing backend social media API endpoints at slash api slash social. Use the existing JWT auth pattern already in place throughout the dashboard.
Rebuild the React client after making these changes so the new section is live. Output all files created and updated.
